## Project Overview
This project evaluates multiple deep learning architectures for automated pneumonia detection using pediatric chest X-ray images.

Models evaluated:
- SmallCNN
- ResNet18
- ResNet50
- EfficientNet-B0
- EfficientNet-B3
- ConvNeXt-Tiny
- ConvNeXt-Small
- Swin-T
- Swin-S
- DenseNet121

## Best Model
EfficientNet-B3 achieved:
- Accuracy: 85.2%
- ROC-AUC: 0.977
- PR-AUC: 0.984
- Recall: 0.997
